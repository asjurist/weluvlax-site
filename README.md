# WeLuvLax repo ready bundle

This folder is set up so you can drop it straight into your repository.

## Main files
- index.html = premium artistic homepage
- shop.html = store homepage with cart drawer and category filters
- stories.html = culture and collection concepts page
- about.html = brand story page
- product_cream_crewneck.html
- product_black_crewneck.html
- product_carolina_blue_crewneck.html
- product_fitted_tee.html
- product_shorts.html
- product_sweatpants.html

## Assets
- assets/css/site.css = shared styling
- assets/js/site.js = cart, filter, reveal, and signup behavior
- assets/js/config.js = place your real email endpoint here
- assets/images = all images needed by the site
- assets/extras = extra mockups from earlier work
- references = older html versions and reference files

## Email signup
To make the signup live, open assets/js/config.js and replace the emailEndpoint value with your real Mailchimp, Klaviyo, Shopify, Formspree, or other endpoint.

## Notes
- The cart is a polished front end demo stored in localStorage.
- If you connect Shopify or another cart later, keep the styling and replace only the cart logic.
- Every worded printed item is presented with the heart and stick logo included in the design direction.
